# Backend Engineering Skills

A complete set of agent skills for building production-grade backend systems from first principles. Language-agnostic — works with Node.js, Python, Go, Java, Rust, or any other runtime.

## Install All Skills

```bash
npx skills add fahad69/backend-engineering-skills --all -g -y
```

## Install Individual Skills

```bash
# The complete guide (recommended starting point)
npx skills add fahad69/backend-engineering-skills@backend-engineering -g -y
```

## Skills Overview

| Skill | Description |
|-------|-------------|
| `backend-engineering` | Complete guide — all topics in one skill |
| `http-protocol` | HTTP methods, status codes, headers, CORS, caching headers, TLS |
| `api-routing` | URL design, path/query params, versioning, route grouping |
| `serialization` | JSON, DTOs, date formats, response envelopes, field handling |
| `auth-authorization` | JWT, sessions, OAuth, RBAC, password hashing, MFA |
| `validation` | Input validation, transformation, sanitization, error messages |
| `architecture` | Controllers, services, repositories, middleware, request context |
| `rest-api-design` | REST principles, CRUD mapping, pagination, error design, OpenAPI |
| `databases` | Schema design, ACID, indexes, transactions, N+1, migrations |
| `caching` | Cache strategies, Redis patterns, TTL, invalidation, HTTP caching |
| `task-queues` | Background jobs, retries, dead letter queues, cron, idempotency |
| `full-text-search` | Elasticsearch, inverted index, relevance, autocomplete, sync |
| `error-handling` | Error types, circuit breaker, retry, fallback, global handler |
| `config-management` | Environment vars, secrets, feature flags, validation at startup |
| `observability` | Structured logging, metrics, alerts, distributed tracing |
| `graceful-shutdown` | SIGTERM handling, request draining, zero-downtime deployments |
| `backend-security` | OWASP, injection, XSS, CSRF, rate limiting, headers, PII |
| `scaling-performance` | Load balancing, horizontal scale, DB replicas, CDN, profiling |
| `concurrency` | Async/await, I/O vs CPU bound, race conditions, deadlocks |

## What These Skills Do

These skills are instruction files for Claude Code agents. When installed, they automatically activate when you ask about backend topics — providing deep, opinionated, language-agnostic guidance on how to build backend systems correctly.

The content is derived from a comprehensive backend engineering course covering all fundamental topics from first principles.
